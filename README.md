## Getting started


To get started, clone the repo and then install the needed gems:

```
$ bundle install
```

Next, migrate the database:

```
$ rails db:migrate
```

Run the test suite to verify that everything is working correctly:

```
$ rspec
```

Run the server

```
$ rails server
```
Try in http://localhost:3000/
